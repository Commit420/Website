# Fake News Detector

![Preview](https://via.placeholder.com/800x400.png?text=Fake+News+Detector+Cyberpunk+Style)

## Übersicht
Der **Fake News Detector** ist ein Cyberpunk-inspiriertes Webtool, das Nachrichten auf ihre Glaubwürdigkeit analysiert. Mit einem futuristischen Design und einer benutzerfreundlichen Oberfläche ermöglicht es schnelle Einschätzungen, ob eine Nachricht potenziell echt oder gefälscht ist.

---

## Features
- **Eingabe**: Nachrichtentext direkt in ein Textfeld eingeben.
- **Analyse**: Das Tool simuliert die Prüfung und gibt an, ob die Nachricht "Echt" oder "Fake" ist.
- **Design**: Modernes Cyberpunk-Design mit dynamischen Effekten.
- **Flexibilität**: Leicht erweiterbar für API-Integration oder andere Datenquellen.

## Installation
1. **Clone das Repository:**
   ```bash
   git clone https://github.com/commit420/fake-news-detector.git
   ```

2. **Kopiere die Dateien in deinen Webserver-Ordner:**
   Für XAMPP-Nutzer:
   ```bash
   mv fake-news-detector /path/to/xampp/htdocs
   ```

3. **Starte den Server:**
   Stelle sicher, dass dein Webserver läuft, und öffne die Seite im Browser unter:
   ```
   http://localhost/fake-news-detector
   ```

---

## Nutzung
1. Öffne die Webseite.
2. Gib einen Nachrichtentext in das Textfeld ein.
3. Klicke auf **Analysieren**.
4. Lies das Ergebnis im Ergebnisbereich (echt/fake).

---

## Anforderungen
- Webbrowser (empfohlen: Chrome oder Firefox)
- Optional: XAMPP oder ein anderer Webserver für lokale Tests
